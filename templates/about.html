About This System

What is it?

The Algorithm RAG Chatbot is a Retrieval Augmented Generation (RAG) system designed to help programmers find the most suitable algorithms for their specific problems. The system combines a vector database of algorithm information with a language model to provide tailored recommendations.

How It Works

When you describe a programming problem, the system:

  1. Converts your query into a vector embedding using Sentence Transformers
  2. Searches the vector database using FAISS to find the most similar algorithm descriptions
  3. Retrieves detailed information about the best matching algorithms
  4. Uses a language model to analyze the matches and generate a recommendation tailored to your specific problem
  5. Formats and presents the recommendation with explanations of why the algorithm is suitable

Technology Stack

This project was built using the following technologies:

Python FastAPI Flask Sentence Transformers FAISS LLM Integration HTML/CSS JavaScript Bootstrap

Features

  • Natural language understanding of programming problems
  • Semantic search for finding relevant algorithms
  • Detailed explanations of why an algorithm is recommended
  • Information about algorithm complexity and use cases
  • Conversation memory to reference previous queries
  • Special commands for system interaction
